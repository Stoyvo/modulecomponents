Description
-----------

``AuroraExtensions_ModuleComponents`` is, as one could imagine, comprised of
components (classes, interfaces, traits) useful to Magento extension developers.

Installation
------------

.. code-block:: sh

    composer require auroraextensions/modulecomponents
